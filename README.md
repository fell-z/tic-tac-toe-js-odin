# Tic Tac Toe made in JavaScript

This is a [project](https://www.theodinproject.com/lessons/javascript-tic-tac-toe)
from The Odin Project.

Concepts practiced in this project are closures,
IIFEs/module pattern and factory functions.

---

There is a similar [project](https://github.com/fell-z/tic-tac-toe-odin)
but made in Ruby earlier in the Ruby on Rails Path.
